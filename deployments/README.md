# Ethersphere Helm Charts

## Quick Start

1. Install [Helm](https://helm.sh) (see [documentation](https://helm.sh/docs/)).
2. Add the repo:

    ```bash
    helm repo add ethersphere https://ethersphere.github.io/helm
    ```

3. Search for charts:

    ```bash
    helm search ethersphere
    ```

4. Update for new releases:

    ```bash
    helm repo update
    ```
